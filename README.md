# Loan_Prediction_ML
Loan Prediction using Machine Learning (Logistic Regression) with data preprocessing and 78% accuracy

Loan Prediction using Machine Learning
This project predicts whether a loan will be approved or not using machine learning techniques
Technologies Used:
Python
Pandas
Scikit-learn

Machine Learning Model:
Logistic Regression

Steps Performed:
Data Cleaning
Handling Missing Values
Feature Encoding(OneHot Encoding)
Train_Test_Split
Model Training
Prediction and Evaluation

Accuracy:
Achieved accuracy of 78%

Dataset:
Loan Prediction Dataset(Kaggle)

Future Improvements:
Improve accuracy using feature engineering
Try advanced model
Hyperparameter tuning
Build a web application using Flask
